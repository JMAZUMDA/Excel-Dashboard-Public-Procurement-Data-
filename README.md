# Excel-Dashboard-Public-Procurement-Data-
India’s public procurement system is a critical driver of infrastructure and socioeconomic development. This project analyzes tender data from Assam for 2021–
2022, focusing on tender types, values, departments, and procurement trends.
Using an interactive Excel dashboard titled “Tender Dashboard Analysis in
Assam,” the analysis highlights key departments, high-value projects, and
procurement challenges. The dashboard aids policymakers, researchers, and
procurement officials in optimizing resource allocation and addressing
inefficiencies.
